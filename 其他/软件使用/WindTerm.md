1. 打开：`安装目录\global\wind.config`
2. 将 `application.lockScreenTimeout` 的值修改为 0 或 负数

# 六、取消登录主密码

1. 打开：`安装目录\save\.wind\profiles\default.v10\user.config`
2. 将 `application.masterPassword` 的值修改为 `false`
# 七、软件中英文切换
### ①、方式一：软件内操作

1. 会话 -> 首选项 -> 设置
2. 外观 -> 语言 -> 选择语言

2. 修改 `application.language` 的值
   1. 中文：`zh-CN`
   2. 英文：`en`
