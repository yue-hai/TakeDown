# 一、简单使用
## 1、下载
`yarn add ant-design-vue`
![image.png](https://cdn.nlark.com/yuque/0/2022/png/29280567/1658718996426-2d324ec6-cb39-44cc-88f5-833271af88f3.png#clientId=u312d8a18-35e5-4&from=paste&height=757&id=uae93b3bd&name=image.png&originHeight=757&originWidth=408&originalType=binary&ratio=1&rotation=0&showTitle=false&size=30893&status=done&style=none&taskId=ua2983199-4786-4bd9-96c1-b0f7c53f8db&title=&width=408)
## 2、修改 main.ts
```tsx
import { createApp } from 'vue'
import App from './App.vue'
// import './index.css'

// Ant Design Vue 全局完整注册
import Antd from 'ant-design-vue'
// Ant Design Vue 样式
import 'ant-design-vue/dist/antd.css'

const app = createApp(App)
// 引用 Ant Design Vue
app.use(Antd)
// 引用他本身
app.mount('#app')
```
