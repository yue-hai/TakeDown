## 2、控制台/作弊模式

### ①、开启控制台/作弊模式

1. 游戏里按回车进入聊天输入页面
2. 输入2遍 `/allowcheats`，就会进入作弊模式
3. 之后按 F10 可以打开关闭控制台

### ②、世界存档关闭作弊模式

1. 游戏主页按 esc -> 设置 -> 基础 -> 打开游戏存档，进入存档目录，具体路径：`C:\Users\用户\AppData\Roaming\Necesse`
2. 进入 `saves/worlds` 目录，找到对应世界名称的压缩文件，<font color="#c00000">将该文件复制到别处作为备份</font>
3. 然后直接进入该压缩文件，不要解压
4. 用记事本打开 `worldsettings.cfg` 文件
5. 搜索 `allowcheats`，将后面的 `true` 改成 `false`，保存关闭即可
6. 如果不能保存，就将 `worldsettings.cfg` 文件复制到别的目录，修改完之后再拖进压缩文件

![](attachments/Pasted%20image%2020251202104833.png)

### ③、人物存档关闭作弊模式

1. 游戏主页按 esc -> 设置 -> 基础 -> 打开游戏存档，进入存档目录，具体路径：`C:\Users\用户\AppData\Roaming\Necesse`
2. 进入 `saves/characters` 目录，找到对应人物名称的 `.dat` 文件，<font color="#c00000">将该文件复制到别处作为备份</font>
3. 然后用记事本打开该 `.dat` 文件

![](attachments/Pasted%20image%2020251202105444.png)

4. 搜索 `cheatsEnabled`，将后面的 `true` 改成 `false`，保存关闭即可

## 3、mod 设置

### ①、More Trinket Slots 增加饰品栏

#### Ⅰ、设置饰品栏命令

```shell
# 命令
/MTS.trinketslots {player/all/online/offline} {set/reset/max/adjust} {新数量}

# 例子：给所有设置饰品栏数量为 6
/MTS.trinketslots all 6

# 例子：给所有设置饰品栏数量为最大（8）
/MTS.trinketslots all max
```

#### Ⅱ、修改配置文件

> 打上这个 mod 后，世界存档内会增加一个配置文件：mtsSettings.cfg

1. 游戏主页按 esc -> 设置 -> 基础 -> 打开游戏存档，进入存档目录，具体路径：`C:\Users\用户\AppData\Roaming\Necesse`
2. 进入 `saves/worlds` 目录，找到对应世界名称的压缩文件，<font color="#c00000">将该文件复制到别处作为备份</font>
3. 然后直接进入该压缩文件，不要解压
4. 用记事本打开 `mtsSettings.cfg` 文件
5. 搜索 `initialSlots`，将后面的数值改为想要修改为的饰品栏数量，比如 `initialSlots = 12,`，然后保存关闭即可
6. 如果不能保存，就将 `mtsSettings.cfg` 文件复制到别的目录，修改完之后再拖进压缩文件
