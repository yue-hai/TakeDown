# 一、功能说明

1. CH341A 编程器（烧录器）是硬件极客、维修人员和嵌入式开发者必备的 底牌 级查故与修复工具。它的核心芯片是 WCH（沁恒）出品的 CH341A，能够实现 USB 到各种串行总线（如 SPI、I2C）的转换
2. 作为一款通用外设，CH341A 编程器的主要功能是通过纯物理的底层连接，直接对主板上的存储芯片进行数据的 读取、擦除、写入、校验，完全绕过操作系统的限制和设备本身的开机自检（POST）
3. 核心应用场景：
	1. 彻底救砖 (Unbricking)：当电脑主板、路由器（如刷错 OpenWrt）、单板计算机（SBC/树莓派等）、电视机顶盒因为刷错固件、断电导致 BIOS/Bootloader 损坏，设备完全无法通电亮机时，强行注入原厂固件使其起死回生
	2. 固件提取与备份 (Dumping)：在维修或逆向工程前，免开机将设备原厂的 Flash 内容完整地提取出来，作为安全备份
	3. 第三方固件强刷：突破官方软件界面的限制，强行刷入破解版 BIOS、解除白名单、或者写入 Coreboot/第三方 UEFI 面板
	4. 底层配置修改：修改主板底层存储的 MAC 地址、SN 序列号，或清除由于密码遗忘导致锁死的主板 BIOS 硬件密码（常见于某些早期笔电）

# 二、硬件说明

> 一套完整且安全的 CH341A 通用烧录套装，通常由以下四个核心硬件模块组成，以应对不同的主板环境

## 1、CH341A 编程器主板

1. 这是整个系统的核心大脑，一头是连接电脑的 USB 接口，另一头是带有一个黑色金属拉杆的 ZIF (零插拔力) 锁紧卡座
2. 分区域设计：ZIF 卡座通常分为左右两个半区（黑色主板上会有明显丝印）：
	1. `25XX` 区域 (SPI 协议)：靠近 USB 接口的那一半。市面上 99% 的电脑主板、路由器、开发板的 BIOS/固件芯片，都属于 25 系列。必须插在此区域
	2. `24XX` 区域 (I2C 协议)：靠近金属拉杆的那一半。常用于极其小容量（几KB）的 EEPROM 芯片，如存储网卡参数、显示器 EDID 的芯片
3. 物理防呆提示：主板印有芯片半圆缺口的示意图，组装时所有模块的第一脚（Pin 1）均需严格顺应缺口方向

![|275](attachments/Pasted%20image%2020260518133632.png)

## 2、1.8V 电压转换座（极其重要/防烧毁硬件）

1. 市面上大部分廉价款 CH341A 编程器的默认输出电压是 3.3V 甚至 5V
2. 为什么必须有它：近几年的新型电脑主板、苹果电脑、高端 ARM 开发板，其选用的芯片变得极其娇贵，工作电压全部降为了 1.8V（芯片型号通常带有 JW 或 W 等代表低压的后缀，如 25Q64JW）
3. 作用：插在主板与烧录夹中间，起到安全降压的功能。如果无视 1.8V 芯片，直接将编程器裸夹上去，瞬间的高压击穿会导致主板存储芯片直接冒烟报废。 它是通用维护中最核心的安全屏障

![|288](attachments/Pasted%20image%2020260518133820.png)

## 3、SOP8 免拆夹 (烧录夹)

1. 形似一个黑色的夹子，尾部连着灰色的细排线和一个蓝色的转接小板
2. 用途 (ISP在线编程)：用于直接咬合主板上被焊死的四边八脚贴片芯片（SOP8 封装）。使用者无需动用电烙铁和热风枪，夹住引脚即可实现物理通讯
3. 方向标识：灰色排线上通常有一根 红颜色的线。这根红线永远代表 第一脚 (Pin 1) 的传输路径，从头到尾必须确保与主板芯片的圆点标识和转接座的缺口标识严格对齐

![|450](attachments/Pasted%20image%2020260518133841.png)

## 4、空白转接板与排针 (拆焊备选方案)

1. 这是免拆夹方案失败时的物理保底工具（由一块绿色空PCB板和两排金属针组成）
2. 何时使用：如果在实际维修中遇到芯片被厂家点胶封死、主板元器件过于密集导致夹子无从下口，或者引脚氧化极为严重导致夹子反复读取失败
3. 使用方式：需使用热风枪将主板上的 BIOS 芯片物理吹下，手工焊接到此绿色转接板上，插上排针后，作为一个标准插件插回 CH341A 编程器的底座上进行离线烧录

![|200](attachments/Pasted%20image%2020260515092932.png)

# 三、连接方式

## 1、说明

1. CH341A 的硬件组装采用层层叠加的模块化设计。只要牢记一个最高通用原则：找准 Pin 1 (第一引脚)，任何主板都可以安全连接
2.  绝对安全红线：
	1. 被施救（被夹取）的主板/路由器，全程必须处于 **完全断电** 状态！（必须拔掉主板的电源适配器、HDMI 线等可能带电的线缆）
	2. 在夹子稳固咬合主板芯片之前，烧录器的 USB 端 **绝不能** 插在电脑上

## 2、识别目标芯片与方向

### ①、被夹取的主板芯片

1. 在主板上找到目标芯片，通常是 8 个引脚的黑色贴片，表面印有 Winbond / Macronix / 25Qxx 字样
2. 由于芯片不能插反，必须先寻找 第一引脚 (Pin 1) 的标识：
	1. 看坑点/白点：芯片表面的四个角中，通常有一个角印有一个微小的圆形凹坑或白点，靠近它的那个引脚就是 Pin 1
	2. 看缺口：若无圆点，芯片一侧的边缘中间有时会有一个半圆形的凹槽缺口。缺口朝向上方时，左上角的第一个引脚即为 Pin 1
3. 查看芯片是 25xx 系列还是 24xx 系列：这个芯片上写着 `25Q64JWSIQ`，前缀的 25，在电子元器件命名的通用规则里，就是代表它是 25 系列的 SPI NOR Flash 芯片

```shell
winbond
25Q64JWSIQ
2520
```

4. 24XX 和 25XX 的区别：
	1. 24XX 系列（I2C 协议）：一般叫 EEPROM。它的容量非常小（只有几KB），速度也很慢。在主板上，它通常用来存储网卡的 MAC 地址、显示器的 EDID 参数，或者极其简单的状态标记。它装不下哪怕一张小图片
	2. 25XX 系列（SPI 协议）：一般叫 SPI Flash。它的容量大得多（几MB 到 几十MB），读取速度极快。所以全世界 99% 的电脑主板、路由器、开发板的 BIOS/UEFI 固件，全都是存在 25 系列芯片里的

![|450](attachments/Pasted%20image%2020260515094415.png)

### ②、CH341A 编程器

1. 锁紧座分为左右两个区域，右下角有白色文字标识， 左半部分为 `25XX`，右半部分为 `24XX`
2. 对于绝大部分固件刷写任务，必须将组件插入 `25XX` 引脚区域
3. 右下角的白色文字标识中，右侧（尾部）有一个半圆形标识，表示起点在右侧（尾部），即第一引脚在右上角

![|425](attachments/Pasted%20image%2020260518134449.png)

### ③、1.8V 电压转换座

1. 绿色电路板上的排针处有标的 1、4、5、8 的数字，该数字即为排针编号，将标号为 1 的排针插入 CH341A 编程器 的 第一引脚处即可
2. 蓝色锁紧座上有 16 个插口，侧边绿色电路板上有标的数字 1、4，该数字即为插口编号，将 SOP8 免拆夹的第一引脚插入此处标号为 1 的插口中即可
3. 关于 SOP8 免拆夹连接 1.8V 电压转换座 的位置：
	1. 如果 SOP8 免拆夹只有 8 个排针，那么只将这 8 个排针插入蓝色锁紧座靠近扳手的这半个部分（即有编号的部分）即可
	2. 如果 SOP8 免拆夹有 16 个排针，那么将其插满即可
	3. 只要保证 SOP8 免拆夹的第一引脚对准蓝色锁紧座的 1 号插口即可
4. 蓝色卡座的上下半区分别代表什么：
	1. 上半区（顶面对齐区）：在芯片烧录领域有一个通用法则，叫 顶面对齐（Top-Justified）。所有芯片的 Pin 1 都要去找卡座的 Pin 1。大部分主板 BIOS 刚好是 8 针封装（SOP8）。所以它插上去之后，恰好只占满了上半区的 4 排孔位。这些孔位在电路板底层连接着旁边那颗黑色的电平转换芯片，负责把 CH341A 送来的 3.3V 降压成 1.8V
	2. 下半区（大封装扩展区）：有些企业级设备，用到容量极其巨大的 SPI Flash 芯片时，8 个引脚不够排布线路，会采用 16 针封装（SOP16） 的长条形芯片。如果遇到那种长条芯片，就需要把这 16 个孔全部插满
	3. 总结：在这个 1.8V 转换座上，下半区不是另外一种协议，而是额外的扩展，如果用 8 针 芯片时，下半区直接悬空无视即可

![|400](attachments/Pasted%20image%2020260515110037.png)

### ④、SOP8 免拆夹

1. 夹子排线：夹子尾部的灰色排线上，绝缘外皮被涂成红色的那一根线，代表 Pin 1 的信号路线
2. 转接小板侧：带有排针的蓝色（或黑色）小板上，通常会在某一个引脚旁印有白色的数字 1。排线的“红线端”插头必须对应插入带 1 的这一侧

# 六、常见报错与排查


## 1、报错：找不到编程器 / CH341A Not Found

1. 现象：软件最下方提示 CH341 Not found 或无任何响应
2. 排查原因：
	1. 驱动装错：Windows 下你可能装成了串口驱动（CH341SER）。必须重装并口/编程器驱动（CH341PAR）
	2. USB 供电不足：台式机请务必插在主板后面的 USB 接口，不要插在机箱前面的 USB 面板或劣质的 USB 扩展坞上

## 2、报错：芯片未响应 / 读出全 `FF` 或 `00`

1. 现象：点击检测（Detect）后，显示提示 Unknown（未知芯片），或者读取出来的 ID 代码全都是 FF FF FF 或 00 00 00
2. 排查原因：
	1. 夹具虚接（占 80%）：免拆夹的金属触点没有精准咬合芯片引脚。表现为夹偏了，或者主板芯片引脚表面存在氧化层。对策：拔下 USB，松开夹子，稍微用力左右扭动几下让金属触点刮开氧化层，重新夹正再试
	2. 供电被主板吸走：有些主板的电源拓扑设计极其坑人，当编程器通过夹子给 BIOS 芯片供电时，电流会分散流向主板上的其他电容和 CPU。对策：如果反复夹了十几次都认不出，说明触发了主板的电流保护，此时只能放弃免拆夹，动用热风枪把芯片拆下来使用“绿板空底座”离线烧录
	3. 防呆引脚插反：返回检查第一脚是否对齐，红线是否接反

## 3、报错：写入成功，但校验失败

1. 现象：擦除和写入进度条都顺利走完，但在最后一步正在校验时报错，提示某一个地址的数据不一致
2. 排查原因：
	1. 忘记点 **清空**：在写入新数据前没有将旧数据清空（Erase），导致新旧代码重叠串键。请勾选清空后重刷
	2. 接触不良导致的杂音：夹子虽然勉强认出了芯片，但在长时间（两三分钟）的高速数据写入时由于微小的振动导致某一个比特写错了。对策：手指捏紧夹子保持不动，重新执行一次自动编程
	3. 芯片开启了底层写保护开关 (Status Register)：部分芯片寄存器设定了强力写保护机制，需要在软件里先手动点击 **取消保护 / Unprotect**

# 七、进阶用法与避坑常识

## 1、MAC 地址与机器码的保留

1. 知识点：绝大多数正规电脑厂商（如联想、华硕、戴尔）和路由器厂家，他们官网提供的 `BIOS .bin` 只是一个通用升级核心包。而每台机器独一无二的 SN 序列号、网卡的 MAC 物理地址、甚至 Windows 原厂正版激活密钥 (OEM Key)，都是在出厂时单独烧录在 BIOS 芯片的某几个特定空白区块里的
2. 避坑警告：如果上来什么都不管，直接拿网上的固件进行全盘 清空 + 刚性覆盖写入，主板虽然能开机，但会发现——系统无法激活、网卡 MAC 地址变成了全是 00 导致无法上网、鲁大师查不出主板序列号
3. 极客正确做法：
	1. 救砖前，一定先点 **读取 (Read)**，把坏掉的“尸体”固件备份下来
	2. 使用专业的极客软件如 UEFITool 或 HxD Hex Editor，或者路由器的 Breed 固件工具
	3. 从备份的固件中，搜索并提取出 MAC 地址块和 SN 码数据
	4. 把它们 缝合/移植 到从官网下载的纯净固件中，最后再将这个 合成固件通过 CH341A 烧录进主板。这就是俗称的 拼合 BIOS 技术

## 2、电压转换的“红线” (1.8V 杀手)

1. 许多人在网上低价买了一个二手的普通 3.3V 编程器，听说能救砖就直接夹在新型主板上，结果伴随着极其微小的一股焦糊味，主板彻底报废
2. 牢记口诀：但凡是 2020 年以后出产的高端 X86 笔记本、高性能 ARM 开发板（如本次的 Radxa O6N）、旗舰平板电脑。在夹上去之前，绝对要拿放大镜看一眼芯片表面型号是否带 `_1.8V / JW / W`，如果是，必须加接 1.8V 绿色转换头后再上电测试

## 3、关于“免拆夹”的使用寿命

1. 免拆测试夹（SOP8 烧录夹）属于易损耗材。夹子内部的 8 根铜质小探针长时间使用后，会发生金属疲劳、弯曲失效或严重的氧化黑化。
2. 当发现某一天无论如何都无法成功识别芯片（且确认主板设计无干扰）时，说明夹子的探针可能已损坏，这个时候只需花几块钱单独购买在排线前端的一个新夹子替换即可
